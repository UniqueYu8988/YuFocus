## 风险

| 风险 | 影响 | 处理方式 | 回退方式 |
|---|---|---|---|
| 误删真实材料包 | 丢失已生成资料 | 删除相关行为先验收、先备份 | 从独立备份恢复 `data/materials` |
| 泄露密钥或 Cookie | 账号和服务安全风险 | 文档只写位置和类型，不写值 | 立即移除泄露内容并轮换密钥 |
| 后台队列重复制作 | 浪费 API 和生成重复数据 | 验证同一 BV 去重和队列恢复 | 暂停后台自动化，手动清理重复项 |
| 大文件重构引入回归 | 核心制作或阅读失败 | 先加验收和测试，再小步拆分 | 用 Git 回退本次代码改动 |
| 历史文档误导方向 | 恢复已退出产品路线 | 以 PRODUCT 和当前代码为准 | 更新或归档过时文档 |
| 继续追旧数据差异 | 消耗时间但不服务未来真实工作 | 标记为旧版遗留，暂不修复 | 回到产品分级重新判断 |
| 工作流被绕过 | 新任务可能跳过定界、任务文件或验证 | 后续任务严格从 `AGENTS.md` 和 `CURRENT_STATE.md` 开始 | 回到 AGENTS 和 CURRENT_STATE 修正入口 |
