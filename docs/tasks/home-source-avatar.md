# 首页最近视频显示 UP 主头像

状态：已完成

## 目标

把简洁首页“最近视频”列表左侧的摄影机图标，替换成对应 UP 主头像，让不同来源的视频更容易区分。

## 范围

- 只修改首页最近视频列表的展示。
- 使用已经保存的固定来源头像 `face` 字段。
- 头像不可用时保留原摄影机图标兜底。
- 不改变登录方式、不新增下载或转写行为、不把头像缓存到本地。

## 风险

- 某些旧固定来源可能没有头像地址，需要兜底。
- 远程头像加载失败时不能让列表出现破图。
- 首页仍必须保持“只更新元数据，不自动入队、下载、转写”的安全边界。

## 验收

- [x] 无头像或图片加载失败时，显示摄影机图标兜底。
- [x] 首页安全检查仍通过。
- [x] TypeScript 类型检查通过。
