# 首页顶部信息精简

状态：已完成

## 目标

精简首页顶部 UI：删除说明文字和四个统计卡片；右上角只保留一个极简刷新图标。

## 范围

- 删除首页标题下方说明文案。
- 删除固定来源、最近视频、等待处理、已有资料四个统计卡片。
- 删除“更新数据”和“进入任务队列”文字按钮。
- 保留一个图标刷新按钮，功能仍为手动刷新视频元数据。

## 不做

- 不改变视频来源、加入队列或任务队列逻辑。
- 不自动下载、转写或制作资料。
