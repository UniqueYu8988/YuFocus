# 任务：简洁数据首页

创建日期：2026-06-20
状态：已完成

## 1. 目标

新增一个简洁首页，作为桌面端默认入口。首页显示来源、最近视频、队列和资料数据，并支持用户手动更新视频元数据。

首页更新只允许：

- 读取本地注册表、队列和资料清单；
- 请求 B 站来源视频元数据并合并到 `data/registry`；
- 更新首页显示时间和错误状态。

首页更新不允许：

- 自动加入任务队列；
- 下载视频或音频；
- 启动字幕获取、本地转写或字幕清洗；
- 触发后台“立即检查”；
- 自动删除或改写资料包。

## 2. 背景

当前默认入口是字幕流水线，信息和操作较多。项目已经具备稳定的 `data/registry`、队列和资料清单，适合增加一个只负责“看数据、手动更新”的轻量入口。

## 3. 本次范围

- 新增 `home` 工作区视图并设为默认入口。
- 新增独立首页展示组件。
- 新增只读读取本地视频注册表的 IPC，首页首次打开不联网。
- 首页“更新数据”只刷新已固定来源的视频元数据。
- 展示最近视频、来源数、待处理队列数和已有资料数。
- 增加静态保护检查，防止首页接入队列保存、制作、下载或转写入口。
- 完成真实桌面端回归。

## 4. 明确不做

- 不修改 Python 生产流程。
- 不自动刷新、自动入队或自动制作。
- 不新增依赖。
- 不重构 `WorkspacePane.tsx` 的其他业务。
- 不修改现有字幕流水线、输出、流程和设置页面行为。

## 5. 验收标准

- [x] 桌面端启动后默认显示“首页”。
- [x] 首页首次加载只读本地数据，不触发 B 站联网刷新。
- [x] 点击“更新数据”后更新已固定来源视频，并显示更新时间。
- [x] 最近视频按发布时间排序，不重复 BV。
- [x] 更新前后队列数量和材料包数量不变。

## 7. 风险

- 错把“更新数据”接到自动化检查，导致自动入队。
- 首页首次加载意外联网。
- 首页共享工作台状态时触发队列保存。
- 多来源返回同一 BV 时显示重复。
